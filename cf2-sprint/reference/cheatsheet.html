考前速记：公式 · 结论 · 流程卡

全课程的压缩包：15 条必背公式、六因素表、30 条必背结论、三道大题的答题流程卡、考场提醒。支持打印（⌘/Ctrl+P）；闭卷考试，建议用默写模式自测——先回忆，再点击揭示。

一必背公式表（闭卷，全部默写）

主题公式
MM Ⅰ（无税）VL = VU = VA
MM Ⅱ（无税）rE = rA + (rA − rD)·D/E
MM Ⅰ（有税）VL = VU + τc·D　（永久债税盾现值 = τc·D）
MM Ⅱ（有税）rE = rA + (1−τc)(rA − rD)·D/E
WACCWACC = (E/V)·rE + (D/V)·rD·(1−τc)
股权价值（永续）E = (EBIT − rD·D)(1−τc) / rE
资产回报率rA = EBIT·(1−τc) / VU　（分母必须是 VU！）
自由现金流FCF = EBIT(1−τc) + 折旧摊销 − 资本支出 − ΔNWC（+ 最后处置年份的资产销售税后收入）
年金现值PV = (C/r)·[1 − (1+r)−T]
常增长（Gordon）/ 残值PV = C1/(r−g)；残值 TVT = CFT+1/(r−g)，还要再贴现回 0 时刻
APVAPV = VU（FCF 按 rA 贴现）+ 税盾现值（rD 或 rA 贴现）
加杠杆隐式方程（通解）V′ = VU / (1 − τc·w)　（w = 目标债务率 D′/V′；新发债 = w·V′ − 旧债）
期权平价定理c + X/(1+r)T = p + S0　（连续复利版：c + X·e−rT = p + S0；有股利：左边 + D）
期权支付call 多头 max(ST−X, 0)；put 多头 max(X−ST, 0)；空头取负号
远期价格F = S0·(1+r)　（+ FV 持有成本 − FV 持有收益）
二项式复制解 Δ·Su + B(1+r) = cu，Δ·Sd + B(1+r) = cd；价格 = Δ·S0 + B
配股除权价P′ = (N·P0 + M·S)/(N+M)；配股权价值 = P′ − S
两版平价公式完全等价，做题以题目的利率口径为准：给「非年化期间利率」用离散版；给连续复利才用 e−rT 版。往届考卷给过公式表但符号写法可能不同——公式表只能救急、不能依赖。

三必背结论 30 条

  1. MM 命题Ⅰ实质：投资决策决定公司价值，融资决策决定价值的分配（不是「市场有效」）
  2. 无税 MM 世界：WACC 恒定、与资本结构无关；有税：税后 WACC 随杠杆上升而下降
  3. rA 与 WACC（必考）：rA 与资产负债比例无关；税后 WACC 与资产负债比例有关
  4. 税盾：每 1 元永久债务创造 τc 元税盾价值；税率越高，举债动力越强
  5. 财务危机成本三类：直接（中介费用）、间接（经营损害）、代理成本；事前全部由股东承担
  6. 股东-债权人代理成本三类型：过度风险投资（资产替代）、投资不足、过度投资；「高管超高薪酬」不属于此类
  7. 权衡理论解释不了：盈利强的公司杠杆低、发股公告股价下跌——这两个靠优序理论解释
  8. 优序融资顺序：内部融资 → 高等级债 → 低等级债 → 可转债 → 普通股
  9. 公司分配只有两种：现金股利和股份回购；送转拆股不是分配
  10. MM 股利无关：完美市场中投资者可「自制股利」，股利政策不影响价值
  11. 美国股利特征：有粘性；管理层关注股利变化而非水平；增派利好、削减利空
  12. 回购三动机：税收优势（低税率+递延）、传递低估信号、储备股权激励用股
  13. 问答核心数据：美国 1997 年后回购超过股利；A 股年分红约 2 万亿 vs 回购约 1 千亿
  14. A 股不回购的根本原因：企业家不认为股价被低估（+ 半强制分红制度 + 大股东偏好现金）
  15. A 股半强制分红：2004 年起分红与再融资资格挂钩（三年不分红丧失资格）；A 股分红需股东大会批准，美国仅需董事会
  16. 美国 IPO：包销最主流；折价发行；择时；规模越大费用率越低
  17. A 股 IPO 分水岭：2019 科创板前是实质审批制（审发行价格、审盈利能力）；注册制都不审，投资者自行判断
  18. A 股 IPO 折价程度：2001 年以来全球名列前茅
  19. WACC 法贴出来的是：公司价值！减去债务等才是股权价值（点名的高频错误）
  20. 收购估值贴现率：用目标公司的 WACC 贴现目标公司的 FCF（风险匹配原则）
  21. APV 税盾贴现：债务总量固定用 rD、负债比率固定用 rA；拿不准时 rD 永远不算错
  22. 期权六因素：S₀、σ、T（美式）、r 升 → call 升；X、股利升 → call 降；put 方向相反（σ、T 同向都升）
  23. 投资者要求的收益率（β/CAPM）：与期权价值无关；CAPM 不能给期权定价（收益非正态）
  24. 波动率上升：call 和 put 价值都上升（下行有底、上行无限）
  25. 套利方向：卖高估、买低估；四笔交易 + 现金流表验证（＋利润, 0, 0）
  26. 美式期权：无分红美式 call 不应提前行权、价值 = 欧式 call（可转债提前转股同理非理性）；美式 put 无解析解
  27. 期权视角：股票 = 以 D 为执行价的看涨期权；债券 = 无风险债 − put；可转债 = 纯债 + call（票息低 ≠ 融资便宜）
  28. 无协同的多元化：降低 σ → 降低股权价值（期权视角），价值转移给债权人
  29. 董事会最重要权力：任免高管；外部治理四件套：资本市场、法律、产品市场、经理人市场
  30. 代理问题主角：中国 = 大股东 vs 中小股东；美国 = 管理层 vs 股东

四计算题与问答题 · 答题流程卡

计算题①（期权套利，20 分）：默写平价公式 → 算 pfair（或 cfair）→ 与市价比较判断高/低估 → 写四笔交易（卖贵买便宜）→ t=0 净现金流 = 套利利润 → 到期两情形现金流表，验证（利润, 0, 0）
计算题②（估值/资本结构，20 分）：① E = (EBIT−rDD)(1−τc)/rE → ② V = E + D → ③ 税盾 = τcD，VU = V − τcD → ④ rA = EBIT(1−τc)/VU → ⑤ WACC（验算 EBIT(1−τc)/V）→ ⑥ 若改资本结构：解隐式方程 V′ = VU + τc·w·V′ → 新 D、新增发债 → 新 rE（有税 MMⅡ）→ 新 WACC → 股东财富变化 = 新增税盾
问答题（A 股回购 vs 股利，20 分）：现象（数据对比：2 万亿 vs 1 千亿；美国 1997 分水岭）→ 成因①估值判断（企业家不认为低估，减持佐证——最根本）②半强制分红制度 ③税收悖论（税制利好回购却没人做）④大股东股权结构 → 对比美国（低估判断 + 股东回报文化）→ 结论句。点睛句：「企业家行动是最真实的市场信号」

五考场提醒

  • 单选题「选最正确的那个」，每题有且只有一个答案；覆盖每一章，不要押题不全。
  • 问答题只答自己班老师的题！答错班级的题目记 0 分。
  • 闭卷：平价公式、MM 公式、WACC 公式必须默写，不指望考卷给公式。
  • 画图题：先标坐标轴（横轴 ST、纵轴收益）、执行价 X、期权费截距，再画折线。
  • 计算题写清步骤和公式再代数——过程分很重要。
