平价定理与套利：拿下计算题①

全卷分值最高的 20 分就在这里。历届计算题①只换数字不换套路——学完这课，你要能闭卷默写「平价公式 → 判高低估 → 四笔交易 → 现金流表」全流程。

计算题① · 20 分历届原题级复现先试后看设计

一欧式期权平价定理（Put-Call Parity）

平价定理 · 离散利率版（考题常给「非年化」期间利率，直接用）c + X/(1+r)T = p + S0
连续复利版（完全等价，题目给连续复利才用）c + X·e−rT = p + S0

适用条件：欧式期权、相同标的、相同执行价 X、相同到期日 T。r 为对应期限的无风险利率。若标的在期内分红，公式左边加股利现值 D：c + X·e−rT + D = p + S0。

公式拆解：这条式子到底在说什么

符号含义怎么理解
c欧式看涨期权现价「涨了跟涨」的权利，今天值多少钱
X/(1+r)T执行价 X 的现值今天存入这么多钱，到期恰好变成 X——一张「保底 X」的无风险债券
p欧式看跌期权现价「跌破 X 可按 X 卖出」的保险，今天的保费
S0标的股票现价—

公式的读法：左边 =「call + 保底钱」——跌了债券到期给你 X 保底，涨了行权跟涨；右边 =「股票 + 保险」——涨了股票跟涨，跌破 X 用 put 按 X 卖出保底。两个组合到期都值 max(X, ST)，是同一个东西的两种做法，所以今天必须同价。

  • 为什么 X 用无风险利率贴现：债券面值 X 是确定支付，确定的钱只能按无风险利率折现，且 r 必须与期限 T 匹配（题给 6 个月利率就用一期 1+r，不要年化折半或平方）。
  • 离散 vs 连续：X/(1+r)T 与 X·e−rT 都是「X 的现值」，只是复利口径不同，两版完全等价——以题目给的利率口径为准。

移项就是合成——四种资产任取三种可复制第四种：

想合成什么移项结果操作
股票S0 = c − p + X/(1+r)T买 call、卖 put、存入 X 的现值（受限不能直接持股的机构这样「持股」）
看涨期权c = p + S0 − X/(1+r)T买 put、买股票、借入 X 的现值
看跌期权p = c − S0 + X/(1+r)T买 call、卖空股票、存入 X 的现值
记忆口诀：「看涨带保底 = 看跌抱股票」（c + PV(X) = p + S₀）。套利题的本质就一句：市价偏离哪一边，就卖那一边、买另一边的合成品——四笔交易正是「卖高估的它 + 买它的三件套复制品」。

无套利证明（证明题可考，三步逻辑要会讲）

  1. 组合 A = 1 份欧式 call + 面值 X 的无风险债券（现值 X/(1+r)T）。到期价值：ST≤X 时 = X；ST>X 时 = (ST−X) + X = ST。
  2. 组合 B = 1 股标的股票 + 1 份欧式 put。到期价值：ST≤X 时 = ST + (X−ST) = X；ST>X 时 = ST。
  3. 两组合到期现金流在任何情形下完全相同 → 无套利要求当前价值必须相等；否则买便宜的、卖贵的，可实现零成本无风险套利。
复制含义：四种资产（call、put、股票、无风险资产）任取三种可复制第四种。给定其中三个价格，第四个的「应有价格」就被锁死——这正是套利题的原理。

二套利题标准模板（20 分大题 · 四步）

母题（2024 课堂考试）：6 个月到期、执行价 $30 的欧式看涨期权价格 $2.5；标的股价 $29；6 个月无风险利率 2.0%（非年化）。同标的、同期限、执行价 $30 的欧式看跌期权价格 $3.0。是否存在套利机会？如有，写出套利策略、交易细节与现金流。

✍️ 建议先在纸上自己算，再逐步展开对照——这才是长在自己身上的 20 分。

1用平价公式算理论价
pfair = c + X/(1+r) − S0 = 2.5 + 30/1.02 − 29 = 2.5 + 29.41 − 29 = 2.91

平价公式把四种资产绑在一起：给定 call、股票、债券三个价格，put 的公平价格就被锁死为 2.91。

2与市价比较，判断高低估

市场 put 报价 3.0 ＞ 理论价 2.91 → put 被高估 0.09。

原则：卖高估、买低估。等价判断：c + X/(1+r) = 31.91 ＜ p + S0 = 32.0，右边贵 → 卖右买左。

3写出四笔交易（卖贵组合、买便宜组合）
  1. 卖出一份看跌期权（+3.0）
  2. 卖空一股标的股票（+29）
  3. 买入一份看涨期权（−2.5）
  4. 买入面值 $30、6 个月到期的无风险债券（−30/1.02 = −29.41）

t=0 净现金流（套利利润）= 3.0 + 29 − 2.5 − 29.41 = +0.09（恰好 = pmarket − pfair，立即落袋）。

4到期现金流表：证明零风险（过程分所在）
交易t = 0到期 S_T ＜ 30到期 S_T ≥ 30
卖出 put+3.0−(30−S_T)0
卖空股票（买回归还）+29−S_T−S_T
买入 call−2.50+(S_T−30)
买入债券（到期收面值）−29.41+30+30
合计+0.0900

今天稳赚 0.09，到期无论股价如何现金流为 0——无风险套利成立。答题时逐格写出这张表并验证合计行 =（＋利润，0，0）。

三大易错：① 四笔交易一笔都不能少、方向不能错；② 现金流表必须每行每列写全并验证「合计」为（＋利润，0，0）；③ 若算出 pfair ＞ pmarket（put 被低估），则四笔全部反向：买 put、买股票、卖 call、卖空债券（借入资金）。规律一句话：永远卖高估的那个，同时买它的复制组合。

考场技巧：这 20 分怎么写才不丢分

① 答卷顺序固定五行：默写平价公式 → 代数算理论价 → 一句话判断（「市场 put 高估 0.09，存在套利机会」）→ 列四笔交易（每笔标注现金流正负）→ 画现金流表 + 结论句。

② 结论句模板（最后必须点破）：「t=0 锁定利润 $0.09，到期无论股价高低净现金流均为 0，该利润无风险，套利成立。」——阅卷看的就是你有没有证明「无风险」，只写四笔不写表会丢一半过程分。

③ 三个防坑：利率口径——题给「6 个月利率 2%（非年化）」就直接用 1.02，切勿再年化、折半或平方；若题目说期内分红——平价左边先加股利现值再判偏离；若问的是 call 错价——cfair = p + S0 − X/(1+r)，套路完全一样，只是四笔换主角。

④ 快检验算：t=0 净现金流必须恰好等于 |pmarket − pfair|。对不上说明四笔中有一笔方向或金额写错了——这是 10 秒钟就能做的自查。

⑤ 时间分配：整题控制在 15 分钟内；先写公式字母式再代数字，步骤分才能拿满。

三平价公式的五种问法（防变式）

平价公式五个量 c、p、S0、X、r——任意给四个都能解第五个。往届练习把所有问法都考过：

  • 反推股价：call $10、put $8、X=80、一年期、r=12%：S0 = c + X/(1+r) − p = 10 + 80/1.12 − 8 = 73.43。
  • 反解利率：股价 $50，call $12 与 put $2 执行价均 $48、一年期：12 + 48/(1+r) = 2 + 50 → 48/(1+r) = 40 → r = 20%。
  • 买 put 最大获利：股价跌到 0 时行权收入最大 = X；扣除权费后最大净获利 = X − p（题目说明不考虑期权费则答 X）。
  • 买 call 最大获利：股价上不封顶，获利无限——这正是 call 与 put 的不对称。
  • call 错价套利：由平价推 call 公平价 cfair = p + S0 − X/(1+r)。市场 c ＞ cfair → 卖 call、买股票、买 put、借入 X/(1+r)；c ＜ cfair → 四笔全反向。

四二项式定价：复制组合三步法（备选计算）

例题（课堂测验）：股票现价 $40，3 个月后要么 $45 要么 $35，无风险利率 2%/季度。求执行价 $40 的 3 个月欧式 put 的价值。

1写出两种情形下期权的到期支付

上行 S=45 → pu = max(40−45, 0) = 0；下行 S=35 → pd = max(40−35, 0) = 5。

2用「Δ 股股票 + B 元无风险资产」复制期权支付

解方程组：Δ×45 + B×1.02 = 0 且 Δ×35 + B×1.02 = 5。两式相减：10Δ = −5 → Δ = −0.5（负号 = 卖空半股）；代回 B = 22.06（正号 = 存钱）。

3无套利定价：期权价值 = 复制成本

p = Δ·S0 + B = −0.5×40 + 22.06 = 2.06。逻辑：复制组合与期权到期支付完全相同，今天必须同价，否则可套利。

复制组合法三步：列支付 → 解 Δ 和 B → 价格 = Δ·S₀ + B。Δ 的直觉：Δ =（期权上行支付 − 下行支付）÷（股价上行 − 下行），就是期权支付对股价的「斜率」（对冲比率）——put 的支付随股价上涨而减少，斜率为负，所以复制 put 要卖空股票；B 是把剩余支付配平的无风险头寸。镜像规律：复制 call 的 Δ 为正（买股票）、B 为负（借钱）；复制 put 的 Δ 为负（卖空）、B 为正（存钱）。

call 版对照（往届练习）：股价 $50，2 个月后 $53 或 $48，r=1%（非年化），X=49：cu=4、cd=0 → Δ = 4÷(53−48) = 0.8（买入 0.8 股），B = −38.02（借入）→ c = 0.8×50 − 38.02 = 1.98。

思想的一般化（状态价格套利）：两个经济状态下，任何证券的支付都可由另两个线性无关的证券组合出来。若复制成本 ≠ 市价，就套利——平价套利、二项式、MM 套利本质是同一招：算复制成本 → 卖贵买复制。

五美式期权与 Black-Scholes（概念题）

  • 美式买权（无分红标的）不应提前行权：由平价关系可推 c ≥ S0 − X/(1+r)T ＞ S0 − X（未到期时），持有期权的价值始终高于立即行权的内在价值，最优策略是等到到期——因此无分红美式 call 价值 = 欧式 call。可转债内嵌转股权是美式期权，同理提前转股非理性。
  • 美式卖权无解析解：只能数值方法近似。
  • Black-Scholes：为欧式期权提供解析解，假设股价服从几何布朗运动；N(d₁)、N(d₂) 为累积正态分布。只需知道其地位，不考推导计算。

六期权视角看公司证券（本章应用必考点）

证券期权解读
股票（股东）相当于持有以债务面值 D 为执行价、以公司资产为标的的看涨期权（有限责任：资产＜D 时一走了之）
债券（债权人）相当于持有面值 D 的无风险债券 + 卖出以 D 为执行价的看跌期权
可转债纯债券 + 看涨期权（所以票面利率低于普通债——不是「更便宜的融资」，期权是有价值的，定价是公允的）
认股权证具有期权性质，相当于股票看涨期权

两条推论（都考过）：

  • 财务困境行为：股票是看涨期权 → 波动率越大越值钱 → 困境中股东有动机过度冒险（资产替代）、投资不足——与第 4 课的代理成本呼应。
  • 多元化损害股东价值：多元化经营降低资产波动率 σ，而股票是资产的看涨期权，σ 下降 → 股权价值下降，价值从股东转移给债权人（债务变安全）。无协同效应的多元化损害股东。

高管期权一行算（曾出现在真题中，会算即可）：CEO 六年前获得 500 万股期权、授予时股价 12 美元、现价 18 美元，行权收益 =（市价 − 行权价）× 股数 = (18−12)×500 万 = 3000 万美元。
