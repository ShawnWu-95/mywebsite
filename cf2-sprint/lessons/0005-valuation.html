Lesson 05 · 公司估值

公司估值：拿下计算题②

「公司估值这一章毫无疑问是重中之重。」计算题②就是估值的价格评估——Pet-One 题在 2024 真题、课堂测验、往届资料中反复出现，本课的六步模板必须做到闭卷独立写出。

计算题② · 20 分考频 15–20%Pet-One 连考三届

一两种现金流：自由现金流 vs 资本现金流

资本现金流：归属于股东和债权人的全部现金流。自由现金流 FCF：假设公司全股权融资（无债）情形下产生的现金流，是一个「人造」但应用最广的概念。FCF = 资本现金流 − 利息税盾。

自由现金流（从 EBIT 出发）$$FCF = EBIT\,(1-\tau_c) + \text{折旧摊销} - \text{资本支出} - \Delta NWC\;(+\;\text{资产销售税后收入})$$（资产销售税后收入：最后处置的年份需要加，中间运营的年份不需要加）

计算起点是 EBIT（息税前利润），EBIT×(1−τc) 称为 EBIAT。注意：利息不扣——因为假设全股权融资；税盾的价值在贴现率（WACC）或单独项（APV）中体现，不能两头都算。

二估值框架与残值

两阶段框架：前期（通常 5–10 年）逐年预测现金流；之后进入永续稳定增长期用残值（终值）整体估算。高速增长期设定严禁超过 10 年。残值三种估法：

  • 永续增长模型（最常用）：TVT = CFT+1 / (r − g)。注意分子 CFT+1 应独立预测，而非简单取 CFT×(1+g)；残值对整体估值影响巨大，参数要谨慎。
  • 可比公司法（乘数法）：基于期末公司状态用 P/E、P/B 等倍数。
  • 清算法：仅适用于非永续项目（如明确 30 年后清算），一般不用于持续经营企业。
残值 TVT 是 T 时点的价值，还要再贴现回 0 时刻（两步贴现）。这是计算题经典失分点。

三三种 DCF 方法

方法贴什么现金流用什么折现率备注
WACC 法公司自由现金流 FCF公司 WACC最常用（实务 >80%），计算题②主方法
APV 法无杠杆 FCF + 税盾单列FCF 按 rA 贴；税盾按 rD 或 rA 贴V = VU + PV(税盾)；适合融资副作用复杂的情形（LBO、项目融资）
FTE 法股权现金流股权成本 rE直接得到股权价值 E；了解即可
WACC 法$$WACC = \frac{E}{V} r_E + \frac{D}{V} r_D (1-\tau_c) \qquad V = \sum_t \frac{FCF_t}{(1+WACC)^t} + PV(\text{残值})$$

要点：权重 E/V、D/V 用市值而非账面价值；股权无税盾，rE 不调整；债务利息抵税，用税后成本 rD(1−τc)。

「几乎每一次考试都有少数同学，方法都对，最后告诉我贴现出来的是股权价值——不对！FCF 用 WACC 贴现出来的是公司价值。要得到股权价值，还要减去债务（以及优先股、可转债等）。」

APV 的税盾贴现规则（选择题考点）：债务总额固定不变 → 税盾像债务利息一样确定，用 rD 贴现；负债比率固定不变 → 债务随企业价值波动，税盾风险与企业整体一致，用 rA 贴现。老师原话：「针对债务的利息税盾贴现，你用 rD 贴现，我永远不会认为你是错的——不管是比例不变还是总量不变。这些方法理论上没有高下之分。」记忆：WACC 法调整的是资本成本，APV 法调整的是价值本身。

三种方法理论上无优劣、结论相同；若企业有目标负债比，可假设 WACC 不变以简化；若资本结构大幅变动，需逐年估计（课堂例题显示近似误差极小：87.9 vs 88.08）。

四公式从哪来：六条式子的来路（懂了就不会背错）

Pet-One 题用到的每条公式都源自同一个底座——永续现金流的市值 = 年现金流 ÷ 折现率（PV = C/r）。逐条拆：

1. 股权价值 \(E = \dfrac{(EBIT - r_D D)(1-\tau_c)}{r_E}\)

分子是股东每年实际到手的钱：公司赚 EBIT，先付利息 rD·D（利息在税前扣除），剩下的按 τc 交税，税后全归股东。这笔钱年年如此（永续），股东要求的回报率是 rE，所以市值 = 年现金流 ÷ rE。

2. 税盾现值为什么恰好 = \(\tau_c D\)（约分演示）

$$\text{每年少交的税} = D\cdot r_D\cdot\tau_c \;\Longrightarrow\; \text{永续现值} = \frac{D\cdot r_D\cdot \tau_c}{r_D} = \tau_c\cdot D$$

税盾现金流与债务利息一样确定，所以按 rD 贴现——分子分母的 rD 恰好约掉，结果里不含利率。这就是为什么「每 1 元永久债务创造 τc 元价值」，与债务利率无关。

3. \(r_A = \dfrac{EBIT(1-\tau_c)}{V_U}\)：分母为什么必须是 VU

rA 的定义是「假设公司无债」时资产的回报率。无债时没有利息可扣，公司年现金流恰好就是 EBIT(1−τc)，对应的市值是无杠杆价值 VU——所以分母是 VU。若误用 VL 作分母，算出来的其实是 WACC（Pet-One 里 360/3000 = 12% 正是 WACC，360/2600 = 13.85% 才是 rA）。往届习题版还专门多问一句「当前的 rA 与负债为 0 时的 rA 各是多少」——答案相同（该题数字下均为 12.71%）：rA 由资产经营风险决定、与杠杆无关，正是选择题「rA 与杠杆」考点的数字验证。

4. WACC 的两种算法为什么必然相等（验算的原理）

$$\text{定义法: } WACC = \frac{E}{V} r_E + \frac{D}{V} r_D (1-\tau_c) \qquad \text{现金流法: } WACC = \frac{EBIT(1-\tau_c)}{V_L}$$

股东年现金流 = E·rE（第 1 条），债权人给公司造成的税后成本 = D·rD(1−τc)，两者加总恰好 = EBIT(1−τc)。两边同除以 V 就得到「定义法 = 现金流法」。这就是第 6 步验算的原理：两种算法必须相等，不等说明前面某步算错了——考场上 10 秒自查。

5. 加杠杆隐式方程的通解（一行出答案）

目标债务率 w（即 \(D' = w\,V'\)）时：$$V' = V_U + \tau_c\, w\, V' \;\Longrightarrow\; V' = \frac{V_U}{1-\tau_c\, w}$$

验证：Pet-One　2600/(1−0.4×0.5) = 3250 ✓；19 级版　5775/(1−0.3×0.5) = 6794 ✓。为什么是隐式的：多发的债本身创造税盾 → V 变大 → 50% 对应的债务也变大，互相咬合，必须解方程而不能拿旧 V×50%。背下这个通式，第（ii）问第一步一行写完。

6. 有税 MMⅡ 从哪来（帮助记对公式，不要求会推）

总回报守恒：公司资产 + 税盾每年产出的现金流，必须恰好分给股东和债权人：\(V_U\, r_A + \tau_c D\, r_D = E\, r_E + D\, r_D\)。代入 \(V_U = E + (1-\tau_c)D\) 整理即得 \(r_E = r_A + (1-\tau_c)(r_A - r_D)\cdot D/E\)。记忆点：rE = 资产回报 + 杠杆风险补偿；有税时补偿打了 (1−τc) 折——因为一部分风险由政府（税盾）分担了。无税版就是把 (1−τc) 拿掉。

五Pet-One 标准模板（真题原题，连考三届）

题目：Pet-One 公司：EBIT = 600 万美元/年，所得税率 40%，权益资本成本 15%，永久债务 1000 万美元，债务利率 10%。
（i）假设股价处于均衡状态，公司整体价值、无负债资本成本 rA、WACC 各是多少？
（ii）公司发行新债回购股份，使债务率增至 50%，新债利率仍 10%。需发行多少新债？WACC 变为多少？对股东财富有什么影响？

✍️ 先在纸上完整做一遍两问，再逐步展开核对。第（i）问六步、第（ii）问三步。

第（i）问 · 六步走

1求股权价值 E
$$E = \frac{(EBIT - r_D D)(1-\tau_c)}{r_E} = \frac{(600 - 1000\times 10\%)\times 0.6}{0.15} = \frac{300}{0.15} = \mathbf{2000}\ \text{万}$$

股权是永续现金流：股东每年拿到（EBIT − 利息）×(1−τ)，按 rE 折现。

2公司整体价值 V
$$V_L = E + D = 2000 + 1000 = \mathbf{3000}\ \text{万}$$
3税盾现值
$$\tau_c\cdot D = 40\%\times 1000 = \mathbf{400}\ \text{万}$$

年税盾 = \(D\, r_D\, \tau_c = 40\) 万，永续按 rD 贴现：40 ÷ 0.1 = 400 万。

4无杠杆价值 VU
$$V_U = V_L - \tau_c D = 3000 - 400 = \mathbf{2600}\ \text{万}$$
5资产回报率 rA
$$r_A = \frac{EBIT(1-\tau_c)}{V_U} = \frac{600\times 0.6}{2600} = \frac{360}{2600} = \mathbf{13.85\%}$$

分母必须用 VU——rA 是「假设无债」时的资产回报率，用 V 算出来的是 WACC。

6WACC + 验算
$$WACC = \tfrac{2000}{3000}\times 15\% + \tfrac{1000}{3000}\times 10\%\times(1-40\%) = 10\% + 2\% = \mathbf{12\%}$$验算：\(EBIT(1-\tau_c)/V_L = 360/3000 = 12\%\) ✓

第（ii）问 · 加杠杆三步

1解隐式方程求新公司价值 V′，得新发债额

债务率 50% 即 \(D' = 0.5\,V'\)，方程是隐式的：

$$V' = V_U + \tau_c D' = 2600 + 0.4\times 0.5\,V' \;\Longrightarrow\; V'(1-0.2) = 2600 \;\Longrightarrow\; V' = \mathbf{3250}\ \text{万}$$

D′ = 3250×50% = 1625 万 → 需新发债 = 1625 − 1000 = 625 万。

2新 rE（有税 MMⅡ）与新 WACC
D/E = 1：$$r_E' = 13.85\% + (1-0.4)\times(13.85\% - 10\%)\times 1 = \mathbf{16.16\%}$$$$WACC' = \tfrac{1}{2}\times 16.16\% + \tfrac{1}{2}\times 10\%\times 0.6 = 8.08\% + 3\% = \mathbf{11.08\%}$$（下降——税盾增强）
3对股东财富的影响

= 新增税盾 = 新发债 × τc = 625×40% = +250 万，全部归原股东所有（宣布时点股价上涨体现）。

三大易错（阅卷扣分点）：① 第（ii）问不能拿旧 V×50% 减旧债当新发债——V 本身会因税盾变大，必须解隐式方程；② rA 的分母用 VU，别用 V；③「对股东财富的影响」标准答案只答新增税盾（+250 万）。

考场技巧：这 20 分怎么写才不丢分

① 每步先写字母式再代数字：「E = (EBIT − rDD)(1−τc)/rE = (600−100)×0.6/0.15 = 2000 万」——公式对了即使算错数也能拿大半步骤分；只写数字则一步错全错。

② 三招自查（各 10 秒）：WACC 双算必须相等（加权法 vs EBIT(1−τc)/V）；E + D 必须等于 V；股东财富变化必须恰好 = 新发债 × τc。任何一个对不上，立刻回头找错。

③ 审题三个关键词：「债务率 50%」是 D/V = 0.5 而不是 D/E（D/E=1 是由 50% 推出来的）；「新债利率」若与旧债不同，MMⅡ 里的 rD 要用新利率；「均衡状态」= 市价反映公允价值，是让你放心用永续公式的信号词。

④ 单位与量纲：全程统一「万」或「百万」，百分比换小数再进公式（40% → 0.4）；最后答案带单位，rE、WACC 保留两位小数。

⑤ 时间分配：第（i）问 10 分钟、第（ii）问 8 分钟、检查 2 分钟；若卡在某步，把已知公式全部列出再往下凑——阅卷按步骤给分。

换数字自练（19 级真题版，套路一步不差）

EBIT 900 万、τ 30%、rE 12%、永久债 1000 万、rD 3%，第（ii）问债务率提至 50%、新债利率仍 3%：

✓核对答案（先自己算完再展开）

（i）E = (900−30)×0.7/0.12 = 5075 万；V = 6075 万；税盾 = 300 万；VU = 5775 万；rA = 630/5775 = 10.91%；WACC = 630/6075 = 10.37%。
（ii）V′ = 5775/(1−0.5×0.3) = 6794 万；D′ = 3397 万 → 新发债 2397 万；rE′ = 10.91% + 0.7×(10.91%−3%) = 16.45%；WACC′ = 0.5×16.45% + 0.5×3%×0.7 = 9.27%；股东财富增加 = 2397×30% ≈ +719 万。

六备选问法：宣布发债回购后的股价与回购数量

若题目问「宣布发债回购后股价如何变化、能回购多少股」：① 先算新公司价值 VL = VU + τcD；② 新股价 = VL ÷ 原股数（公告日即涨）；③ 回购股数 = 发债额 ÷ 新股价。数字演示见第 4 课课堂例题（1 万元公司发 5000 债：股价当天 20→24 元，按 24 元回购 208.33 股）。最常见错误：用旧股价 20 元算回购数量（250 股）。

无税对照（概念题精华）：无税 MM 下发债回购，V 不变、股价不变、股东财富不变——发债回购只是价值的重新切分，不创造价值；便宜的债被更高的股权成本抵消，WACC 不变。有税才创造价值，增量 = 税盾，全部归原股东。
